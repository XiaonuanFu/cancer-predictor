## Class Imbalance Review

- Reviewed whether having many more cancer samples than normal samples could affect the prediction model.
- Current dataset balance discussed:
  - tumor samples: `449`
  - normal samples: `41`
  - approximate ratio: `11:1`

- Key conclusion:
  - The model can still learn useful signals, but plain accuracy is not enough because the model may learn to favor the larger tumor class.
  - Balanced accuracy, normal recall, confusion matrix, and PR-AUC are more useful checks.
  - The existing training code already uses `class_weight="balanced"`, which helps reduce bias toward the larger class.

- Important limitation:
  - The current tumor and normal samples come from different sources or processing paths, so results should be treated as exploratory research rather than clinical evidence.

  - 5-year survival:
    - low-risk group: `84.7%`
    - high-risk group: `46.8%`

## Key Idea

The prediction model should become one part of a larger research story.
