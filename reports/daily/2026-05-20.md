## Issues / Notes

- Docker Hub 网络不稳定，构建时曾遇到基础镜像元数据请求超时。
  - 通过 `BASE_IMAGE=bio:latest` 使用本地已有镜像作为构建基底，绕开了 Docker Hub 拉取阻塞。

- ChEMBL 官方 dump 中 `idx_assays_desc` btree 索引因为长文本超过 PostgreSQL btree 单行大小限制而失败。
  - 数据没有丢失。
  - 用同名 hash 索引替代：

```sql
CREATE INDEX idx_assays_desc ON chembl.assays USING hash (description);
```
