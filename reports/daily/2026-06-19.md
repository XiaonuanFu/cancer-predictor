## Main Model Results

- Dataset:
  - tumor samples: `471`
  - adjacent normal samples: `41`
  - shared protein-coding genes: `19,938`
  - model genes after filtering: `19,486`

- Train/test split:
  - train normal: `33`
  - train tumor: `376`
  - test normal: `8`
  - test tumor: `95`

- Model comparison:
  - Random Forest reached `1.0` accuracy and `1.0` balanced accuracy on this test split.
  - Logistic Regression reached about `0.942` accuracy and `0.968` balanced accuracy.
  - Linear SVM reached about `0.942` accuracy and `0.968` balanced accuracy.

Balanced accuracy means the average correctness across classes. It is safer than plain accuracy when one class is much smaller.

## Interpretation

- The TCGA-only adjacent-normal report is more controlled than mixing TCGA tumors with GTEx normal samples, because both tumor and normal labels come from the same TCGA COAD STAR counts source.
- The model results are promising for research, but they should not be treated as clinical proof.
- The key limitation is sample size: the test set has only `8` adjacent normal samples, so one mistake would noticeably change normal-class metrics.

## What Was Learned

- COAD mutation data shows the expected colorectal cancer pattern: frequent `APC`, `TP53`, `KRAS`, and `PIK3CA` mutations.
- Recurrent hotspots such as `KRAS p.G12D` and `BRAF p.V600E` are useful research signals, but they should be checked against external cancer knowledge bases before being described as confirmed driver events.
- Mutation burden means the number of mutation records per sample. In this notebook it is a simple count, not a clinically standardized TMB per megabase.
- VAF means variant allele fraction, the fraction of sequencing reads supporting the mutated allele.
- A very high model score can still be fragile when the test set has very few normal samples.
