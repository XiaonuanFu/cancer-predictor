# COAD Model Reports

Generated model reports are written here when the pipeline runs in the Jupyter container.

Expected generated files:

- `metrics_summary.csv`
- `important_genes.csv`
- `confusion_matrix.png`
- `roc_curve.png`
- `pr_curve.png`
- `model_report.md`

These generated outputs are local research artifacts and are not committed by default.
