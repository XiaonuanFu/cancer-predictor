# Web Design Directory

Use this directory for the design and implementation documentation of the Web code in `web_code/`.

## Suggested Files

- `web-code-design.md`: current Web app architecture, UI structure, API list, and next design tasks.
- `website-design-requirements.md`: English-first website requirements for the COAD project presentation, curated data schema, read-only interactions, and later cloud deployment.
- Future additions can include wireframes, API schemas, UI copy, or frontend test notes.
