# Requirements Directory

Use this directory for requirements and project planning documents.

## Suggested Files

- `product-requirements.md`: current project requirements and scope.
- `cancer-prediction-model-requirements.md`: requirements for the COAD-focused two-stage cancer prediction model plan.
- `coad-research-report-requirements.md`: requirements for turning the COAD work into a broader research report with clinical, molecular, survival, mutation, pathway, and model-interpretation analyses.
- Future additions can include user stories, data analysis requirements, model requirements, report requirements, or release plans.
